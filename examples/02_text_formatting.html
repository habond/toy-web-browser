Text Formatting

This example demonstrates all supported text formatting options.

Basic Formatting

Text can be bold, italic, underlined, strong, or emphasized.

Code Formatting

Use inline code for technical terms and code snippets.

Example: function greet() { return "Hello!"; }

Links and Spans

Links are supported: click here. Generic containers use span elements.

Nested Formatting

You can combine different formatting styles together. Even emphasized code works!

Preformatted Text

Preformatted text preserves whitespace and uses monospace font:

    This is preformatted text
    with preserved    spacing
        and indentation.

    Code example:
    function hello() {
        console.log("Hello, world!");
        return true;
    }

Font Demonstration

Regular text uses Open Sans font.

Bold text uses Open Sans Bold font.

Code text uses Source Code Pro monospace font.

Mixed: regular, bold, and monospace in one line.
